# HHL algorithm implementation

Project created for the Quantum Computing and Machine Learning Seminar.

### Authors
- Wiktoria Marczuk
- Szymon Gogulski

# Project goals

1. Reviewed the HHL algorithm,
2. Implemented the algorithm for a 2x2 system,
3. Compared with the classical solution,
4. Implemented HHL for a 4x4 system.
